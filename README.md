# 💰 한국 연봉 실수령액 계산기

파이썬으로 작성된 간단한 연봉 기반 월 실수령액 계산 프로그램입니다.

## 기능

- 연봉 입력 시 월 실수령액 자동 계산
- 4대 보험 (국민연금, 건강보험, 장기요양보험, 고용보험) 공제
- 소득세 및 지방소득세 계산
- 부양가족 수에 따른 세금 계산
- 상세한 공제 내역 표시

## 사용 방법

```bash
python salary_calculator.py
```

프로그램 실행 후 연봉과 부양가족 수를 입력하면 월 실수령액이 계산됩니다.

## 계산 항목

### 4대 보험
- **국민연금**: 4.5% (상한액 590만원, 하한액 37만원)
- **건강보험**: 3.545%
- **장기요양보험**: 건강보험의 12.95%
- **고용보험**: 0.9%

### 세금
- **소득세**: 누진세율 적용 (6% ~ 45%)
- **지방소득세**: 소득세의 10%

==================================================

## 주의사항

- 이 계산기는 2026년 기준 세율을 사용합니다
- 실제 급여 계산은 회사의 복리후생, 비과세 항목 등에 따라 달라질 수 있습니다
- 정확한 실수령액은 급여 명세서를 확인하시기 바랍니다

## 요구사항

- Python 3.6 이상
